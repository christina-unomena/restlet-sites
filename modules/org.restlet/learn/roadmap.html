Version 3.0

  • Java 7 requirement
  • Web API documentation and management (based on APISpark)
  • Enumerations instead of String constants
  • Direct connector and converter configuration
  • Jetty 9.1 (SPDY / HTTP 2.0, HTTP client)
  • Jackson 2.4
  • Mailing lists migration to Google Groups
  • New tutorial

Conclusion

Due to the open source nature of the Restlet project, the above timeline is subject to change. However, we will do our best to respect it and update it as frequently as possible. Please, feel free to contact us for questions and comments.

If you see a feature you would like to help us with, please jump in and contributeto make it happen sooner.

If you don't have enough time to share, your company can also help us by funding the development of some features. See our co-development offer at Restlet.

Finally, nothing is written in stone, so please express yourself, we are looking forward to your feed-back!
