Version 2.3

  • Java 7 requirement
  • Web API documentation and management
  • Jetty 9.1 upgrade
  • Netty 5.0 integration
  • Vert.x edition
  • WebSocket support
  • Restlet API refactoring
  • JAX-RS 2.0 support
  • OAuth 2.0 support
  • Restlet API enhancements
  • Mustache template extension
  • Distribute client-side profiles of JARs
  • Continuous integration enhancements
  • Revamp documentation
  • Mailing lists migration to Google Groups

For additional informations, please read this detailled 2.3 road map.

  • IDE tooling

Conclusion

Due to the open source nature of the Restlet project, the above timeline is subject to change. However, we will do our best to respect it and update it as frequently as possible. Please, feel free to contact us for questions and comments.

If you see a feature you would like to help us with, please jump in and contribute to make it happen sooner.

If you don't have enough time to share, your company can also help us by funding the development of some features. See our co-development offer at Restlet.
